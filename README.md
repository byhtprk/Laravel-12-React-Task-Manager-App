# Laravel 12 + React Starter Kit

A simple task manager project built with Laravel 12, React, and Inertia.js.

## Installation

1. `composer update`
2. `npm install`
3. Set up `.env` file
4. `php artisan migrate`
5. `php artisan serve` and `npm run dev`

## Features

- Create tasks
- List tasks
- Update tasks
- Delete tasks
- Laravel 12 backend
- React frontend with Inertia.js
